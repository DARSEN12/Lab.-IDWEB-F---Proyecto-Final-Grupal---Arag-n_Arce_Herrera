**Descripción del proyecto:**  
El presente proyecto consiste en el desarrollo de una **página web de adopción** que permite a los usuarios registrarse e iniciar sesión mediante las funcionalidades de **login** y **register**.  

El sistema funciona a través de un **backend desarrollado en Python**, el cual se ejecuta desde la consola accediendo a la carpeta **Backend** y ejecutando el comando:

```bash
python app.py
